첫화면
------------
1. 회원가입
------------
회원가입하면 정보 > 디비로 보낸다
아이디입력
>중복되는 아이디인지확인>디비에서 회원정보 불러와서 일치하는 아이디 있는지 검색
>중복일시, 다시 아이디 입력
>중복아닐시, 아이디입력 성공 > 배열에 저장?
패스워드입력
>패턴에 맞는 패스워드 입력하기
>패턴에 맞지 않을 시 패스워드 다시 입력하기
>패턴에 맞으면 패스워드입력 성공. > 배열에 저장?
이름입력
>이름은 그대로 입력 > 배열에 저장?
주소입력
>주소도 그대로 입력 > 배열에 저장?
전화번호(핸드폰번호)
>패턴에 맞게 저장? > 
>맞지않으면 다시 입력
>맞게 저장했을 시 > 배열에 저장?
이메일
>이메일도 중복검색? >디비에서 이메일을 불러와서 일치하는 이메일 있는지 검색 
>중복검색일 시, 다시 이메일 입력
>패턴에 맞으면 이메일 입력 성공. > 배열에 저장?

총 회원정보입력성공(회원가입성공)
배열에 저장된 입력값 디비로 보낸다.
디비로 저장 성공
>>회원가입성공.
>>입력한 정보 디비에서 불러와서 총입력정보 확인.
>로그인하기

2. 회원주문
---------------
로그인 할 시, 
아이디입력 
>입력한 아이디가 디비에 존재하는지 확인 > 디비에서 아이디 불러와서 일치하는 아이디 있는지 확인하기
>불일치 시, '존재하는 아이디가 없습니다. 다시 입력해주세요' 다시 입력
>입력한 아이디가 일치 시, 비밀번호 입력 화면
>입력성공한 아이디는 변수에저장
비밀번호입력
>입력한 비밀번호를 변수에저장, >디비에서 회원정보를 불러 와 아이디,비밀번호가 일치하는지 확인한다.
>불일치 시, '비밀번호입력이 잘못되었습니다. 다시입력해주세요' > 다시 비밀번호 입력
>비밀번호 일치 시, 로그인 성공
로그인변수에 아이디와 비밀번호 저장?
>>로그인성공
주소확인
로그인변수와 일치하는 회원의 주소 불러오기 > 디비에서 회원정보를 불러와 일치하는 아이디가 있는지 확인
>일치하는 정보에 주소를 불러온다
>주소 출력 ' 주문할 주소 :' + 주소+ '가 맞습니까?'
 
>>메뉴로 넘어감
